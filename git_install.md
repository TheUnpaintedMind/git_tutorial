### Git installation :-
*In order to learn Git & GitHub, installing Git is essential, below are the methods of installation on Windows, Mac and Linux.*

## Mac Installation :-
1) Go to [https://git-scm.com/install/](https://git-scm.com/install/)
2) Select Mac and install Xcode, there are two ways to do it:-
   * Go to Spotlight Search -> Terminal , and type "git", if help menu opens, git is already installed if not then click on install in the pop-up menu.<br>
   * Go to App Store and install Xcode which is an IDE(Integrated Development Environment) , after that restart your terminal(if opened earlier) and type git.<br>
   **Usually Xcode CLI(Command Line Interface) is enough for installing git.**
## Windows Installation :-
1) Go to [https://git-scm.com/install/](https://git-scm.com/install/)
2) Click on Windows and install
3) Run the installer and click on next for default configurations
4) Open Windows PowerShell and type git
5) If git is installed then various git commands and their use cases would be displayed below

## Linux Installation:-
1) Open Terminal
2) Type "git", and see if git commands are displayed
3) If they are not displayed, then git is not installed
4) Type "sudo apt update" to update the list of all local package index
5) Type "sudo apt install git", if git successfully installs then we are good to go
6) If "unmet dependencies" is displayed type "sudo apt --fix-broken install"
7) Finally type "sudo git install" and verify the installation by typing "git --version"
