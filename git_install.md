### Git installation :-
*In order to learn Git & GitHub, installing Git is essential, below gives the methods of installation on Windows, Mac and Linux.*

## Mac Installation :-
1) Go to [https://git-scm.com/install/](https://git-scm.com/install/).
2) Select Mac and install Xcode, there are two ways to do it:-<br><br>
   2.1) Go to Spotlight Search -> Terminal , and type "git", if help menu opens, git is already installed if not then click on install in the pop-up menu.<br><br>
   2.2) Go to AppStore and install Xcode which is an IDE(Integrated Development Environment) , after that restart your terminal(if opened earlier) and type git.<br><br>
   **Usually Xcode CLI(Command Line Interface) is enough for installing git.**
